Monumentos del Mundo

Instrucciones

  1. 1. Desplázate por el mundo manteniendo presionado clic izquierdo y arrastra para navegar.
  2. 2. Mantén el cursor sobre el Pin Rojo para observar el nombre del monumento.
  3. 3. Presiona clic izquierdo sobre el Pin Rojo para observar el monumento.

Instrucciones

  1. 1. Desplázate por el mundo manteniendo presionado sobre la pantalla y luego arrastra para navegar.
  2. 2. Presiona sobre el Pin Rojo para observar el nombre del monumento.
  3. 3. Presiona el botón "Visitar" para observar el monumento.
